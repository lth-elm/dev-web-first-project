Le C#

C# est un langage de programmation orientée objet, fortement typé, dérivé de C et de C++, ressemblant au langage Java3. Il est utilisé pour développer des applications web, ainsi que des applications de bureau, des services web, des commandes, des widgets ou des bibliothèques de classes3. En C#, une application est un lot de classes où une des classes comporte une méthode Main, comme cela se fait en Java3.


Type

    Qu’est ce qu’un Type ?

  • Le type détermine la place mémoire occupée par la variable.
  • Le type détermine les valeurs possibles/acceptables qui pourront être stockées (une à la fois!) dans la variable.

    Les Types Possibles

  • Les Entiers : Byte (1octet), short (2 octets), int (4 octets), long (8 octets).
  • Les Réels : float, simple précision (32 bits) double double précision (64 bits).
  • Les booléens : bool (True False).
  • Les caractères : char, la valeur d’un caractère doit être placée entre deux simples quote (’).
  • Chaîne de caractères : string, est un ensemble de caractères placé entre double quote (").

Les Structures conditionnelles

    Il existe 2 types de structures conditionnelles :

  • La classique : if (si), else if (sinon si), else (sinon).
  • Choix multiple : S’il y a plus de deux choix possibles, l’instruction selon switch permet une facilité l’écriture plutôt qu’une série de if imbriqués.


Les Structures répétitives

    Elles permettent une exécution répétitive(boucle) et cessent lorsqu’une condition de test n’est plus vérifiée!
    Il existe deux familles de structures de répétitions :

  • Itérative : for (pour)
  • Evénementielle : while (Tant que), do ... while (Faire ... Tant que).
